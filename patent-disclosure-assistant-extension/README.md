# 专利技术交底书助手 Chrome 插件

一个智能的浏览器插件，帮助用户快速、规范化地撰写专利技术交底书。基于"发现问题→设计方案→技术效果"的三步法思维，提供完整的文档生成、质量检查和导出功能。

## ✨ 功能特性

### 🎯 核心功能

- **智能表单引导**: 4步式信息收集流程，引导用户完整填写所有必要信息
- **多种文档模板**: 支持默认、详细、简化三种模板类型
- **实时文档预览**: Markdown格式预览，支持分步查看和编辑
- **质量自动检查**: 集成质量审核系统，自动检查文档完整性和规范性
- **多格式导出**: 支持 Markdown、HTML、PDF、纯文本等多种格式导出
- **本地存储管理**: 自动保存草稿，支持文档历史记录管理

### 🚀 高级特性

- **页面内容提取**: 智能从网页提取技术内容
- **性能优化**: 内置缓存、防抖、错误处理等性能优化机制
- **响应式设计**: 适配不同屏幕尺寸，优化移动端体验
- **离线使用**: 无需网络连接，完全本地化运行

## 📦 安装指南

### 方法一：开发者模式安装（推荐）

1. **下载插件文件**
   ```bash
   # 下载整个 patent-disclosure-assistant-extension 文件夹
   ```

2. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入: `chrome://extensions/`
   - 或者点击菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在扩展程序页面右上角，开启"开发者模式"开关

4. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `patent-disclosure-assistant-extension` 文件夹
   - 点击"选择文件夹"

5. **验证安装**
   - 插件图标应出现在Chrome工具栏
   - 点击图标可打开插件界面

### 方法二：打包安装

1. **打包插件**
   - 在扩展程序页面，点击"打包扩展程序"
   - 选择 `patent-disclosure-assistant-extension` 文件夹
   - 点击"打包扩展程序"

2. **安装.crx文件**
   - 将生成的.crx文件拖拽到Chrome扩展程序页面
   - 确认安装

## 📖 使用说明

### 第一次使用

1. **点击插件图标** 在Chrome工具栏点击插件图标
2. **查看欢迎界面** 插件会自动显示使用指南
3. **开始创建文档** 点击"新建文档"开始撰写

### 基本操作流程

#### 第1步：基本信息
- 填写文档标题（必填）
- 输入文档编号（可选）
- 描述技术领域（必填）
- 填写发明人信息（必填）

#### 第2步：背景技术
- 描述现有技术现状（必填）
- 指出现有技术存在的问题（必填）

#### 第3步：技术方案
- 明确要解决的技术问题（必填）
- 详细描述技术解决方案（必填）
- 列出关键技术特征（可选）

#### 第4步：技术效果
- 说明有益效果（必填）
- 提供性能数据（可选）
- 描述具体实施方式（可选）

### 文档管理

#### 保存文档
- 插件支持自动保存草稿
- 点击"保存"按钮可手动保存
- 保存的文档可在"文档管理"标签页查看

#### 加载文档
- 在"文档管理"标签页选择文档
- 点击"编辑"按钮加载

#### 导出文档
- 在"文档预览"标签页点击"导出文档"
- 选择导出格式（Markdown/HTML/PDF/纯文本）

### 质量检查

#### 启动质量检查
- 在"文档预览"标签页点击"质量检查"按钮
- 系统会自动分析文档完整性

#### 查看检查结果
- 质量检查报告包含：
  - 总体评分（0-100分）
  - 错误项（必须修复）
  - 警告项（建议修复）
  - 建议项（可选改进）

2. 点击浮动工具栏的"提取文本"

- **缓存机制**: 自动缓存常用数据，提升响应速度
- **延迟加载**: 按需加载资源，优化内存使用
- **防抖处理**: 避免频繁操作导致的性能问题

## ❓ 常见问题

### Q: 插件无法安装？
A: 请确保：
- Chrome版本 >= 88
- 已启用开发者模式
- 文件夹结构完整

### Q: 文档丢失了怎么办？
A: 插件支持自动保存，请检查：
- 是否开启了自动保存功能
- Chrome存储空间是否充足

### Q: 导出的文档格式有问题？
A: 可以尝试：
- 更换导出格式
- 检查文档内容是否完整
- 重新生成文档

### Q: 质量检查分数很低？
A: 请检查：
- 是否填写了所有必填项
- 内容长度是否足够
- 是否缺少关键技术描述

### Q: 插件运行缓慢？
A: 可以尝试：
- 清理浏览器缓存
- 重启Chrome浏览器
- 检查Chrome扩展程序占用内存

## 🔍 数据存储

### 本地存储
- 插件使用Chrome本地存储保存数据
- 数据仅存储在本地设备上
- 不会上传到任何服务器

### 存储内容
- 文档草稿
- 文档历史记录
- 用户设置
- 质量检查结果

### 数据清理
- 在设置中可选择"清除所有数据"
- 卸载插件时会自动清理数据
