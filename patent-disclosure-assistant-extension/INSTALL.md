# 快速安装指南

## 系统要求

- **浏览器**: Chrome 88+ 或基于Chromium的浏览器
- **操作系统**: Windows 10+, macOS 10.14+, Ubuntu 18.04+
- **权限**: 需要访问本地存储和下载功能

## 安装步骤

### 步骤1: 下载插件

下载整个 `patent-disclosure-assistant-extension` 文件夹到您的计算机

### 步骤2: 打开Chrome扩展管理

在Chrome浏览器地址栏输入以下地址并按回车：
```
chrome://extensions/
```

### 步骤3: 启用开发者模式

1. 在扩展程序页面的右上角
2. 找到"开发者模式"开关
3. 点击启用开发者模式

### 步骤4: 加载插件

1. 点击"**加载已解压的扩展程序**"按钮
2. 在文件选择对话框中，选择 `patent-disclosure-assistant-extension` 文件夹
3. 点击"**选择文件夹**"按钮

### 步骤5: 验证安装

安装成功后，您应该看到：

1. **工具栏图标**: Chrome工具栏出现专利技术交底书助手图标 📄
2. **插件卡片**: 在扩展程序页面显示插件信息
3. **状态指示**: 插件状态为"已启用"

## 快速测试

安装完成后，请按以下步骤测试插件：

1. **打开插件**: 点击工具栏中的插件图标
2. **新建文档**: 点击"新建文档"按钮
3. **填写信息**: 按提示填写基本信息
4. **生成文档**: 完成信息收集后点击"生成文档"
5. **预览查看**: 切换到"文档预览"标签页查看结果

## 常见安装问题

### ❌ 问题1: 找不到"加载已解压的扩展程序"按钮

**解决方案**:
- 确保已启用"开发者模式"
- 刷新扩展程序页面
- 重启Chrome浏览器

### ❌ 问题2: 提示"无法加载扩展程序"

**解决方案**:
- 检查文件夹路径是否正确
- 确保manifest.json文件存在
- 查看Chrome控制台错误信息

### ❌ 问题3: 插件图标不显示

**解决方案**:
- 在扩展程序页面检查插件状态
- 尝试固定插件图标到工具栏
- 重启Chrome浏览器

### ❌ 问题4: 插件功能异常

**解决方案**:
- 检查Chrome版本是否符合要求
- 尝试禁用其他插件看是否冲突
- 查看错误日志：`chrome://extensions/` → 点击"详细信息" → "检查扩展程序错误"

## 卸载插件

如需卸载插件：

1. 打开 `chrome://extensions/`
2. 找到"专利技术交底书助手"
3. 点击"移除"按钮
4. 确认删除

## 权限说明

插件需要以下权限：

- **存储权限**: 保存文档和设置
- **下载权限**: 导出文档文件
- **活动标签页权限**: 提取网页内容

所有数据仅存储在本地，不会发送到任何服务器。
