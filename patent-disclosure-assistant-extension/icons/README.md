# 图标文件说明

这个目录包含Chrome扩展的图标文件。

## 图标尺寸
- icon16.png - 16x16 像素，用于浏览器扩展栏
- icon32.png - 32x32 像素，用于扩展管理页面
- icon48.png - 48x48 像素，用于扩展管理页面
- icon128.png - 128x128 像素，用于Chrome Web Store

## 建议设计
- 使用蓝色主色调（#3498db）
- 简洁的专利文档或技术图标
- 清晰易识别
- 支持透明背景

## 制作工具
可以使用以下工具制作图标：
- Adobe Illustrator
- Figma
- Canva
- 在线图标制作工具

注意：需要将图标保存为PNG格式。
